# Packages and Pricing (Draft)

## Starter

For small clubs or pilot launches.

- Core auth + role-based API
- Opportunities and applications flow
- Basic messaging/media endpoints
- Standard CI and deployment guidance

Commercial model (draft):
- Setup fee + monthly subscription

## Pro

For growing scouting teams.

- Everything in Starter
- Session/device operations
- Extended Postman E2E and quality gates
- Monitoring and alert runbooks
- Priority support window

Commercial model (draft):
- Higher monthly subscription + optional SLA add-on

## Enterprise

For multi-team organizations and high governance requirements.

- Everything in Pro
- Custom integration support
- Security and compliance review package
- Dedicated onboarding and architecture sessions
- Incident response playbook alignment

Commercial model (draft):
- Annual contract + custom scope pricing

## Pilot Offer (Recommended)

- Duration: 2-4 weeks
- Scope: one role journey end-to-end
- Exit criteria:
  - functional acceptance
  - no critical defects
  - agreed rollout plan

# Social Visuals (nextscout.pro)

Bu paket 60 adet sosyal gorsel uretir:
- 20 Instagram (1080x1350)
- 20 Facebook (1200x630)
- 20 X (1600x900)

Marka metni: `nextscout.pro`

## Dosyalar

- `generate_social_images.py` -> 60 SVG gorsel uretimi
- `run_generate_social_images.bat` -> tek tik calistirma
- `../public/assets/social/2026-03/backgrounds/*.svg` -> arka plan gorselleri

## Cikti Konumu

- `public/assets/social/2026-03/instagram/`
- `public/assets/social/2026-03/facebook/`
- `public/assets/social/2026-03/x/`

## Hemen Uret

```bat
cd c:\Users\Hp\Desktop\PhpstormProjects\scout_api_pr_clean\seo_social_gifs
run_generate_social_images.bat
```

## Not

Bu ortamda scripti calistirip dosyalari fiziksel olarak uretme adimini otomatik dogrulayamadim; script yerelde tek komutla uretecek sekilde hazirlandi.
